# AI-Powered Student Success Analytics Platform

## Overview

This project uses Machine Learning to predict student academic performance based on study habits, previous academic records, sleep patterns, extracurricular activities, and practice tests.

The application provides an interactive dashboard built with Streamlit and includes model comparison, feature importance analysis, and personalized recommendations.

## Features

* Student Performance Prediction
* Interactive Streamlit Dashboard
* Multiple Model Comparison
* Feature Importance Analysis
* Personalized Recommendations
* Data Visualization and Analytics

## Technologies Used

* Python
* Pandas
* Scikit-Learn
* Random Forest
* Matplotlib
* Streamlit

## Machine Learning Models

* Linear Regression
* Decision Tree Regressor
* Random Forest Regressor

## Dataset

Student Performance Dataset from Kaggle.

## Key Insights

* Previous Scores are the strongest predictor of performance.
* Study Hours significantly influence academic outcomes.
* Academic history plays a major role in future success.

## How to Run

```bash
pip install -r requirements.txt
python main.py
streamlit run app.py
```
